# mandelbrotPGM
Renders Mandelbrot set to PGM images with C and OpenCL GPGPU.

## Command line usage 
`mandelbrotPGM [width] [height] [precision]`
- **width:** width of generated image in pixels
- **height:** height of generated image in pixels
- **precision:** no. of iterations to perform per pixel
